# Learning Guide: Expense Split + Receipt OCR

## What you learn

1. Real financial workflow modeling
2. Settlement minimization algorithm
3. Full-stack CRUD with validation
4. API-driven analytics
5. OCR-style text extraction logic

## Interview talking points

1. How paid/owes/net balances are computed.
2. How debtor-creditor settlement suggestions are generated.
3. Why split modes need validation (percent total = 100, shares > 0).
4. How OCR parsing confidence is estimated.
5. How to migrate from JSON to PostgreSQL + background jobs.

## Strong upgrades

1. Real image OCR integration (Tesseract/Cloud Vision).
2. UPI-based settlement links.
3. Member auth + invite links.
4. Scheduled reminder notifications.
5. Test suite for split and settlement correctness.
